# vue-preset
结合 vue-cli3 的 preset 搭建基于 git repo 的前端项目模板

## 快速开始

```bash
# 安装 vue-cli 3.0
npm install -g @vue/cli

# 根据远程 preset 创建项目
vue create --preset vue-devin-cli/vue-preset --clone my-project

# 本地预览
cd my-project && npm run serve

```
